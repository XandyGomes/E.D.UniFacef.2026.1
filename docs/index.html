Algoritmos de Ordenação

A ordenação é a base para a eficiência de outros algoritmos. Aqui comparamos o Quick Sort (divisão e conquista) com o Bubble Sort e Selection Sort.

  • Quick Sort: O(n log n) - Eficiente para grandes massas.
  • Bubble/Selection: O(n²) - Didáticos para pequenas listas.
